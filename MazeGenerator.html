Maze Generator


          Version hexagonale          

          Version classique                    

Description

Petit projet personnel réalisé en JavaScript avec la bibliothèque P5.
Il s'agit d'un générateur de labyrinthe dans lequel on peut se déplacer une fois la génération terminée. Dans la version classique, les déplacements se font avec les flêches du clavier et dans la version hexagonale, les déplacements se font avec le pavé numérique :

  • le 4 déplace vers le coin en bas à gauche
  • le 5 vers le bas
  • le 6 vers le coin en bas à droite
  • le 7 vers le coin en haut à gauche
  • le 8 vers le haut
  • le 9 vers le coin en haut à droite
